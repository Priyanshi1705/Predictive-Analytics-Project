# Predictive Analytics Using Historical Data

## 📌 Project Overview

This project demonstrates how historical data can be used to build a predictive analytics model. Using the Advertising dataset, a Linear Regression model is trained to predict future sales based on advertising expenditure across TV, Radio, and Newspaper channels.

The project includes data preprocessing, model training, performance evaluation, and prediction visualization.

---

## 🎯 Objectives

- Build a predictive model using historical data.
- Clean and preprocess the dataset.
- Train a Linear Regression model.
- Evaluate model performance.
- Visualize actual and predicted values.

---

## ✨ Key Features

- Data cleaning and preprocessing
- Predictive modeling using Linear Regression
- Model evaluation using R² Score
- Visualization of prediction results

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Priyanshi1705/Predictive-Analytics-Project.git
```

Navigate to the project folder:

```bash
cd Predictive-Analytics-Project
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Project

Navigate to the source folder:

```bash
cd src
```

Run the Python script:

```bash
python prediction.py
```
